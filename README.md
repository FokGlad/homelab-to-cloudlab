# From DevOops to DevOps

A Devops journey documenting the migration from a self-hosted homelab to modern cloud infrastructure.
This repository will track my architecture and networking decisions, my failures, fixes and lessons learned.

The goals are simple :
Transition from a homelab to a hybrid-cloud based infrastructure while applying real-world DevOps practices and learning by breaking things (on purpose)

Scope :
  - VPS provisioning and hardening
  - VPN-based connectivity
  - Reverse proxies, DNS, and TLS
  - Mail, auth, and service exposure
  - Infrastructure as Code
  - Docker and Kubernetes
  - Gradual decommissioning of on-prem services
